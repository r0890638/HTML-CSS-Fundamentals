Now let's add some photos on this webpage. To do so, we need to use img tag.

While most of the HTML tags, which once opened <(tag)> needs to be closed with a forward slash "<(/tag)>", there are self closing tags. That means these tags do not need to be closed with a forward slash.

To insert an image, we use the img tag, which is a self-closing tag. For example: let us use this code, img.

The code img is a self-closing tag.

The image that you see here is by using the following code:

< img src="https://i.imgur.com/3VTKgE9.png" alt="Computer User"> Here img is a self-closing tag. While alt is an attribute that provides alternative information for an image if a user for some reason cannot view it, width is an attribute that specifies the width of the element, in pixels.
